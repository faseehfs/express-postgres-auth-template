# Express

This is a minimal Express application which demonstrates user authentication. It uses MySQL.

## Convensions

### General

- Max line length: 80 characters.

### API

- Accepts and returns JSON most times.
- If the process was successful, returns a JSON containing `message`.
- If the process fails, returns a JSON containing `error`.

## Notes

- You should install `mysql2`, not `mysql`. Run `npm install mysql2`. Click [here](https://sidorares.github.io/node-mysql2/docs) to view the docs.
